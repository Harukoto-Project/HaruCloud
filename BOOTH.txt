【HaruCloud Sync v1.0.0（正式版）】

■ ひとことで

自宅や VPS 上の MinIO（S3 互換）に、PC 上のフォルダをワンクリックでアップロード同期できる、Windows 向けデスクトップアプリです。

クラウドストレージではなく自分で持つ MinIO に、複数フォルダを folder_id ごとに整理して送りたい人向けです。


■ こんな方におすすめ

・すでに MinIO を運用していて、Windows PC から手軽にオブジェクトを足したい
・複数の同期セット（フォルダ A → folder_id X、フォルダ B → folder_id Y）を GUI で管理したい
・ブラウザや CLI だけだと手間なので、ローカルアプリで接続テストと同期まで済ませたい


■ 機能

・MinIO 接続設定 … endpoint、リージョン、アクセスキー、シークレット、bucket、TLS の ON/OFF
・同期対象 … folder_id とローカルパスを複数行で登録（フォルダは配下すべて、単一ファイルは親フォルダをルートとしてそのファイルのみ）
・接続テスト … バケットへの疎通確認
・同期（アップロード） … 差分判定（MD5 と ETag）で変更分のみアップロード（進捗表示あり）
・設定の保存 … Windows のローミング AppData に保存（再インストール後もパスは環境により異なります）
・GitHub 経由のアプリ更新（設定に応じて）

※アップロード中心の同期です。MinIO から PC への自動ダウンロード同期などは今後の検討事項です。


■ 動作環境

・OS：Windows 64bit
・ランタイム：Microsoft Edge WebView2（多くの環境で既に入っています）


■ 同梱・入手後の流れ

（1）配布されているインストーラー（.exe）を実行してインストール
（2）アプリを起動し、MinIO の情報と同期対象を入力
（3）設定を保存 → 接続テストで確認 → 同期でアップロード


■ ご利用上の注意

・重要データ・本番バケットを扱う前にバックアップを推奨します。
・MinIO 側のリージョン設定と、アプリに入力する region が一致していないと認証エラーになります。
・API が HTTP のサーバーには、TLS をオフにするか http:// でエンドポイントを指定してください（HTTPS との取り違えで接続に失敗することがあります）。
