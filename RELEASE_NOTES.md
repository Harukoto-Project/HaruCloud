# HaruCloud Sync v1.0.0

**正式版（stable）**の初回リリースです。

## 新機能・改善

### 同期対象（フォルダ／単一ファイル）

- ローカルパスに **フォルダ** を指定した場合は、従来どおり配下のファイルを再帰的に同期します。
- **ファイル** を指定した場合は、その **親フォルダをルート** とみなし、**その1ファイルだけ**を同期します（キーは親からの相対パス）。

### MinIO 同期（差分アップロード）

- **HEAD** とローカル **MD5**／リモート **ETag** で変更のないオブジェクトはスキップします。
- 進捗は照合フェーズとアップロードフェーズの二段表示です。

### アプリ更新

- GitHub の `latest.json` を参照する **Tauri アップデーター**、起動時の自動確認、手動の更新確認／インストール。

### その他

- MinIO エラーの詳細表示とヒント、同期ボタンの連打防止、NSIS インストーラー、設定の AppData 保存、MIT License。

## 動作環境

- Windows x64
- Microsoft Edge WebView2

## インストール

配布の **NSIS インストーラー**（`HaruCloud Sync_1.0.0_x64-setup.exe`）を実行してください。
