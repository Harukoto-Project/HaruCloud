# HaruCloud Sync v0.1.0-beta.2

**ベータ版**です。本番データを扱う前にバックアップと検証環境での動作確認を推奨します。

## 新機能・改善

### MinIO 同期（差分アップロード）

- 同期時にオブジェクトを **HEAD** で確認し、ローカルファイルの **MD5** とリモートの **ETag** が一致するファイルは **アップロードをスキップ**します。
- 進捗表示は **照合フェーズ**（全ファイルの確認）と **アップロードフェーズ**（変更分のみ）の二段になります。
- ログに **アップロード件数 / スキップ件数** を表示します。変更がなければ「差分なし（すべて一致）」となります。
- **注意:** マルチパートでアップロードされたオブジェクトなど、`ETag` にハイフンが含まれる場合は一致判定できないため **再アップロード**されます。

### アプリ更新（GitHub Releases）

- **Tauri アップデーター**を組み込み、GitHub の `latest.json` を参照して更新を確認します。
- 起動時に **自動で更新の有無を確認**します（未設定・オフライン時はログに出さず終了します）。
- 画面から **「更新を確認」「更新をインストール」** を実行できます。インストール後は再起動がかかります。
- 配布者向けのビルド手順は **README** の「アプリ更新（GitHub Releases）」を参照してください。

### その他（これまでのベータラインに含まれる主な内容）

- MinIO 接続・同期の **エラー表示の詳細化**（`DisplayErrorContext`）と、TLS/HTTP の取り違え・リージョン不一致時の **ヒント文言**。
- **同期ボタンの連打防止**と進捗 UI。
- **NSIS インストーラー**、設定の **ローミング AppData** 保存。
- **MIT License**、BOOTH 向け紹介文（`BOOTH.txt`）。
